# Лента сообщества и тарифы

## Лента сообщества

Mini App показывает публичные генерации как визуальную мозаику, а не линейный список.

Доступны:

- витрина наиболее интересной работы;
- фильтры «Для вас», «Новое», «Фото» и «Видео»;
- лайк и дизлайк с повторным нажатием для снятия реакции;
- автоматическое снятие противоположной реакции;
- серверное ранжирование, где дизлайки уменьшают итоговый score;
- публичная карточка автора без раскрытия Telegram ID;
- профиль автора с количеством публичных работ, лайков и дизлайков;
- галерея работ автора;
- постоянная ссылка на каждый пост в формате `?post=<task_id>`;
- создание собственной работы на той же модели и с тем же форматом, но со своим текстом.

Telegram-лента использует те же серверные реакции и показывает кнопки лайка, дизлайка, профиля автора, создания своей версии и прямую ссылку на пост.

## Защита авторского текста

Исходный prompt автора не входит в публичный API ленты и не передаётся в Mini App. Это серверное ограничение, а не только визуальное скрытие.

При выборе **«Создать своё»** переносятся только безопасные параметры: модель, формат, качество и длительность. Пользователь пишет собственный prompt. Исходный текст автора не записывается в пользовательский FSM и не показывается в Telegram.

## Хранение реакций

Лайки сохраняются в `feed_likes`, дизлайки — в `feed_dislikes`. На одну работу у пользователя может быть только одна реакция. При установке лайка существующий дизлайк удаляется, и наоборот.

Таблица `feed_dislikes` создаётся идемпотентно через compatibility schema SQL и входит в обычный Alembic/staging rollout.

## Профиль автора

Публичный профиль использует имя, username при наличии и публичный `partner_code`. В API ленты не возвращаются Telegram ID и prompt пользователя.

## Добавление тарифа администратором

На главном экране админки и в разделе «Каталог» доступна кнопка **«Добавить тариф»**. Она запускает существующий девятишаговый мастер создания `CreditPackage`:

1. уникальный код;
2. название;
3. фото-кредиты;
4. видео-кредиты;
5. универсальные кредиты;
6. цена;
7. длительность подписки;
8. признак безлимита;
9. публичные условия.

После сохранения тариф включается и появляется в пользовательском каталоге, если выдаёт кредиты либо корректный временный безлимит.
