# Библиотека референсов

Раздел «Мои референсы» предназначен для повторного использования уже загруженных фотографий.

## Поведение

- библиотека собирается из отдельных пользовательских загрузок, а не из прошлых промптов;
- одинаковые Telegram `file_id` не дублируются;
- показываются до 100 последних уникальных изображений;
- фото и изображения, отправленные документом, поддерживаются отдельно;
- библиотека отображается одной медиа-карточкой с кнопками «назад/вперёд»;
- пользователь может отметить несколько изображений и снять выбор;
- после выбора референсов пользователь заново выбирает модель;
- промпт, формат и качество из старой генерации не переносятся;
- количество переданных изображений ограничивается актуальным лимитом выбранной модели.

Кнопка «Ещё вариант (тот же реф)» остаётся отдельным сценарием полного повтора генерации и может восстанавливать прежние настройки.
